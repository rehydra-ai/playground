# Rehydra Playground

A playground to test out [Rehydra](https://rehydra.ai) — a PII (Personally Identifiable Information) Anonymization tool.

## About

This interactive playground allows you to experiment with Rehydra's anonymization capabilities. Enter text containing sensitive information and see how Rehydra detects and anonymizes PII in real-time.

## Setup

Make sure to install the dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```
